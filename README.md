# DIGIY Projet Maison

**Transforme une idée en cahier clair.**

DIGIY Projet Maison est un assistant maison de cadrage projet.  
Il aide une personne qui ne connaît pas forcément le web à transformer une idée simple en dossier clair : cahier des charges, pages nécessaires, fonctionnalités, base de données conseillée, prompt IA et plan de production.

---

## Vision

Beaucoup de personnes disent :

> Je veux un site.  
> Je veux vendre en ligne.  
> Je veux une application.  
> Je veux présenter mes services.

Mais elles ne savent pas toujours expliquer :

- quelles pages il faut créer ;
- quelles fonctions sont vraiment utiles ;
- quelles données doivent être stockées ;
- ce qui doit être public ;
- ce qui doit rester privé ;
- ce qui est prioritaire ;
- ce qui peut attendre.

**DIGIY Projet Maison prépare le terrain avant la construction.**

---

## Doctrine

> On ne copie pas l’ancien outil.  
> On construit notre assistant de cadrage projet maison, inspiré du besoin terrain.

Ce projet est une création originale DIGIY / UFDOS.  
Il ne reprend aucun code source privé d’un autre outil.

L’objectif est de bâtir un assistant simple, autonome et utile pour les novices du web, les entrepreneurs, les commerçants, les artisans, les agences et les porteurs de projets.

---

## À quoi sert l’outil ?

L’outil permet de :

- choisir un type de projet ;
- décrire une activité ;
- indiquer la ville, le public visé et l’objectif ;
- cocher les besoins importants ;
- importer éventuellement des fichiers HTML existants ;
- analyser rapidement un template ;
- générer un cahier des charges ;
- générer un prompt IA ;
- générer une base SQL conseillée ;
- exporter le résultat en texte, Markdown, SQL ou JSON.
