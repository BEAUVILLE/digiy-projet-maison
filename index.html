Transforme une idée en cahier clair.

Un outil maison pour aider les novices du web à expliquer leur projet. Tu choisis le métier, tu ajoutes les besoins, tu peux importer un template HTML, et DIGIY prépare une feuille de route propre.

On ne copie pas l’ancien outil. On construit notre assistant de cadrage projet maison, inspiré du besoin terrain.
Commencer le cadrage
1
Dire le métier

Commerce, restaurant, immobilier, taxi, beauté, BTP, réservation ou projet libre.

2
Ranger l’idée

L’outil clarifie les pages, les actions, les données et les priorités.

3
Lire le template

Ajoute des fichiers HTML pour repérer menus, formulaires, titres et images.

4
Sortir le dossier

Cahier des charges, prompt IA, SQL conseillé, plan de production.

Cadrage Sortie

1. Informations du projet

On commence simple. Le but n’est pas de parler ingénieur. Le but est de faire sortir une route claire.

2. Besoins à activer

Coche ce qui compte. L’outil va préparer les blocs utiles sans surcharger.

3. Template HTML existant

Optionnel. Tu peux ajouter des fichiers HTML pour que l’outil lise les titres, menus, formulaires, images et boutons.

Lecture locale seulement. Rien n’est envoyé à un serveur. Les fichiers sont analysés dans ton navigateur.

Aucun fichier ajouté pour le moment.

4. Dossier généré

Le dossier va sortir ici.Remplis le projet puis clique sur “Préparer le projet”. L’outil te donnera un cahier clair, un prompt IA, une base SQL conseillée et une synthèse exploitable.
